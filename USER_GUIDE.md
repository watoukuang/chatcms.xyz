# AiTodo.Me 使用指南

## 🎯 产品定位
AiTodo.Me 是一个基于 AI 的智能时间规划工具，帮助你高效管理任务和时间。所有数据保存在浏览器本地，无需后端，支持数据导入导出。

## 📋 三大核心功能

### 功能一：制定任务（AI 任务拆解）

**位置**：首页 `/`

**功能说明**：
- 输入你的目标和可用时间窗口
- AI 自动将大目标拆解为可执行的小任务
- 生成详细的时间安排表
- 拆解后的任务可移动到固定任务栏

**使用步骤**：
1. 选择开始时间和结束时间（定义时间窗口）
2. 可选：输入目标总时长（分钟）
3. 点击「智能提示词模板」选择合适的模板
4. 在输入框中描述你的任务需求，替换模板中的 `[方括号]` 内容
5. 点击发送按钮或按 Enter 键
6. AI 会返回任务拆解结果和时间安排

**智能提示词模板**：
- ⚡ **快速拆解任务**：适合日常工作任务拆解
- 🍅 **番茄工作法**：按 50分钟专注 + 10分钟休息的节奏规划
- 📅 **周计划制定**：制定本周工作计划
- 💼 **会议准备**：准备会议材料和 PPT
- 📚 **学习计划**：制定学习进度安排

**示例输入**：
```
请帮我拆解以下任务：
目标：撰写项目周报和准备周会PPT
时间窗口：2025-11-17 09:30 ~ 12:00
截止时间：2025-11-17 15:00
约束条件：10:30-11:00需要参加短会

请输出：
1. 任务分解列表（含预计时长、优先级）
2. 时间安排表
3. 风险与缓冲建议
4. JSON格式数据
```

### 功能二：固定任务（任务看板）

**位置**：固定日程 `/schedule`

**功能说明**：
- 查看和管理固定的周任务看板
- 支持 24 小时时间轴（可配置为 8 小时工作时段）
- 可添加、修改、删除任务
- 支持重复任务设置

**使用步骤**：

#### 1. 配置工作时段
- 点击右上角「工作时段」按钮
- 选择工作日（周一至周日）
- 设置每日工作时长（默认 8 小时）
- 设置开始时间（默认 09:00）
- 可选：启用午休时间（默认 12:00-13:00）
- 点击「保存配置」

#### 2. 查看任务看板
- 周视图：显示本周一到周日的任务
- 时间轴：按小时显示，支持跨时段任务
- 任务卡片：显示任务内容、状态、时间

#### 3. 添加任务
- 点击「+ 新增任务」或点击空白时间格
- 填写任务信息：
  - 任务日期
  - 开始时间、结束时间
  - 任务内容
  - 备注（可选）
  - 状态（待处理/进行中/已完成/已延期）
- 点击「提交」

#### 4. 编辑/删除任务
- 点击任务卡片进入编辑模式
- 修改信息后点击「提交」
- 或点击删除按钮删除任务

**任务状态说明**：
- 🟡 **待处理**：尚未开始的任务
- 🔵 **进行中**：正在执行的任务
- 🟢 **已完成**：已经完成的任务
- 🔴 **已延期**：超过截止时间未完成

### 功能三：灵活备选（候选任务池）

**位置**：灵活备选 `/blocklog`

**功能说明**：
- 管理备选任务列表
- 智能评估哪些任务可以插入固定任务日历
- 避免时间冲突
- 支持优先级排序

**使用步骤**：

#### 1. 添加备选任务
- 在备选任务页面点击「添加任务」
- 填写任务信息：
  - 任务标题
  - 预计时长（分钟）
  - 截止时间（可选）
  - 优先级（高/中/低）
  - 依赖任务（可选）

#### 2. 智能插入
- 使用智能提示词模板「智能插入任务」
- 输入评估时段和插入策略
- AI 会分析固定任务日历的空闲时段
- 给出可插入建议和替代方案

**示例输入**：
```
我有一批备选任务，请帮我找到合适的时间插入：
评估时段：2025-11-18 ~ 2025-11-22
插入策略：优先高优先级，仅空闲>=60分钟

候选任务：
1. 标题：代码review，时长：90分钟，截止：2025-11-20，优先级：高
2. 标题：学习新技术，时长：120分钟，截止：无，优先级：中
3. 标题：整理文档，时长：60分钟，截止：2025-11-22，优先级：低

请输出：
- 可插入建议（时间、理由）
- 冲突说明与替代方案
- JSON格式数据
```

#### 3. 移动到固定任务
- 根据 AI 建议的时间
- 在固定任务页面手动添加
- 或使用 JSON 数据批量导入

## 💾 数据管理

### 数据导出

**位置**：右上角「数据管理」按钮

**导出格式**：
1. **JSON 格式**：完整数据备份，可用于恢复
2. **CSV 格式**：Excel 表格格式，方便查看和分析

**导出步骤**：
1. 点击「数据管理」
2. 选择「导出 JSON」或「导出 CSV」
3. 文件自动下载到本地

**文件命名**：
- JSON: `aitodo_backup_YYYYMMDD_HHmmss.json`
- CSV: `aitodo_tasks_YYYYMMDD_HHmmss.csv`

### 数据导入

**导入步骤**：
1. 点击「数据管理」
2. 选择「导入数据」
3. 选择之前导出的 JSON 文件
4. 确认导入
5. 页面自动刷新加载新数据

**注意事项**：
- 导入会覆盖现有数据，建议先导出备份
- 只支持 JSON 格式导入
- 数据格式必须正确

### 清空数据

**清空步骤**：
1. 点击「数据管理」
2. 选择「清空数据」
3. 确认操作（不可恢复）
4. 页面自动刷新

⚠️ **警告**：清空操作不可恢复，建议先导出备份！

## 📝 最佳实践

### 1. 任务拆解技巧
- **明确目标**：清晰描述你要做什么
- **设定时间**：给出合理的时间窗口
- **列出约束**：说明已有安排和限制条件
- **选择模板**：使用合适的提示词模板
- **迭代优化**：根据 AI 建议调整计划

### 2. 时间管理建议
- **配置工作时段**：先设置你的工作日和工作时间
- **固定任务优先**：重要的会议、截止任务先放入固定任务栏
- **留出缓冲**：不要把时间排满，预留 20% 缓冲时间
- **定期回顾**：每周回顾任务完成情况
- **及时调整**：根据实际情况调整计划

### 3. 数据安全
- **定期备份**：建议每周导出一次 JSON 备份
- **多设备同步**：通过导出/导入在多设备间同步
- **版本管理**：重要变更前先导出备份
- **云端存储**：将导出文件保存到云盘

## 🔧 常见问题

### Q: 数据保存在哪里？
A: 所有数据保存在浏览器的 localStorage 中，不会上传到服务器。

### Q: 换浏览器或清除缓存会丢失数据吗？
A: 是的，清除浏览器数据会导致数据丢失。建议定期导出备份。

### Q: 可以在多个设备上使用吗？
A: 可以，通过导出/导入功能在设备间同步数据。

### Q: AI 拆解的任务可以修改吗？
A: 可以，AI 只是提供建议，你可以根据实际情况调整。

### Q: 支持团队协作吗？
A: 目前不支持，这是个人时间管理工具。

### Q: 提示词模板可以自定义吗？
A: 当前版本使用内置模板，你可以在输入框中自由修改。

## 🚀 快速开始

1. **首次使用**：
   - 点击「工作时段」配置你的工作时间
   - 在首页尝试使用智能提示词模板
   - 将拆解的任务添加到固定任务栏

2. **日常使用**：
   - 早上：查看今日固定任务
   - 工作中：根据实际情况调整任务状态
   - 晚上：规划明天的任务

3. **周度规划**：
   - 周日晚上：使用「周计划制定」模板规划下周
   - 周五下午：回顾本周任务完成情况
   - 导出数据作为周报素材
